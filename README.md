# Agentur-PM-Kostencheck – Excel-Sofortvorlage

Öffentliche Verkaufs- und Wissensseite der **Viista-Products GmbH** für den
Kostenvergleich von Projektmanagement- und Agentursoftware.

## Digitales Produkt

Die Excel-Sofortvorlage vergleicht bis zu fünf konkrete Software-Angebote und
berechnet automatisch:

- Zwölfmonatskosten je Angebot
- Mehrkosten gegenüber dem günstigsten eingetragenen Angebot
- Effektivkosten je Nutzer und Monat
- ein Vergleichsdiagramm

Preis: **9,90 EUR einmalig**. Nach erfolgreicher Zahlung wird die Datei
automatisch bereitgestellt. Das Produkt richtet sich ausschließlich an
Unternehmer; es ersetzt keine Funktions-, Vertrags-, Rechts- oder Steuerprüfung.
